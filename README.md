# Kiraradesk - 在桌面上养一只粒子粉色小章鱼

原项目：[https://github.com/Caster-binying/fishdesk](https://github.com/Caster-binying/fishdesk)

## ✨ 功能特点
### 🎨 视觉改进
粉色章鱼形态：触手根部粗壮、尖端纤细，贴近真实章鱼形态

无规律触手摆动：8条触手各自独立随机摆动，具有不同的波动频率和相位

平滑颜色渐变：身体和触手都有平滑的颜色过渡和透明度变化

动态响应：章鱼会根据鼠标接近程度改变波动幅度和透明度

### 🎮 交互功能
三种交互模式：

逃避模式：章鱼会远离鼠标

跟随模式：章鱼会跟随鼠标移动

无视模式：章鱼忽略鼠标，自由游走

智能边界处理：平滑转向，避免撞击屏幕边缘

鼠标接近响应：触手波动增强，透明度变化

### ⚡ 性能优化
优化点分布算法，减少渲染点数

降低帧率至30FPS，更平滑的动画效果

改进物理模拟，更自然的运动轨迹

### 🚀 快速开始
安装依赖

```
# 基础依赖
pip install pygame numpy

# Windows系统额外依赖（用于透明窗口）
pip install pywin32

```
运行程序

```
python FISHdesk.py
```

🎯 操作方法
键盘控制
ESC - 退出程序

1 - 切换为逃避模式

2 - 切换为跟随模式

3 - 切换为无视模式

鼠标交互
章鱼会对鼠标位置做出反应（取决于当前模式）

鼠标接近时，触手波动会增强


## ⚙️ 主要参数配置
形态参数
`OCTOPUS_BODY_RADIUS` - 身体半径

`OCTOPUS_TENTACLES` - 触手数量

`TENTACLE_LENGTH` - 触手长度

`TENTACLE_BASE_WIDTH` - 触手根部宽度

`TENTACLE_TIP_WIDTH` - 触手尖端宽度

颜色参数
`BODY_COLOR_START` - 身体起始颜色

`BODY_COLOR_END` - 身体结束颜色

`TENTACLE_COLOR_START` - 触手起始颜色

`TENTACLE_COLOR_END` - 触手结束颜色

运动参数
`PET_SPEED` - 基础移动速度

`ROTATION_SPEED` - 转向速度

`WANDER_STRENGTH` - 随机游走强度

交互参数
`MOUSE_INTERACTION_RADIUS` - 鼠标交互半径

`MOUSE_ATTRACTION_STRENGTH` - 鼠标吸引力度

`MOUSE_REPULSION_STRENGTH` - 鼠标排斥力度

## 🔧 主要改进（相对于原项目）
1. 触手形态改进
添加了触手根部到尖端的宽度渐变

改进了触手点分布算法，使触手更贴近身体

添加了每条触手的独立弯曲参数

2. 摆动系统升级
每条触手有独立的波动频率和相位

添加了三种波动模式：基础波动、扭动波动、摆动波动

波动幅度从根部到尖端逐渐减小

3. 交互体验优化
改进了鼠标交互算法，响应更自然

添加了多种交互模式选择

优化了边界转向处理，运动更平滑

4. 性能提升
减少了渲染点数至10000点

降低了帧率至30FPS，减少CPU占用

优化了向量化计算，提高渲染效率


### 🐛 已知问题
在某些Linux系统上可能无法正确显示透明窗口

高性能模式下CPU占用率可能较高

触手数量较多时可能会出现轻微的性能问题
